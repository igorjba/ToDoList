# Rodar Front

1. Instalar dependencias
```
npm install
```

2. Na raiz do projeto, onde tem o arquivo db.json executar, para simular o backend
```
json-server db.json -m ./node_modules/json-server-auth
```

3. executar projeto, na raiz do projeto
```
npm run dev
```
